### Generating the documentation with Doxygen

Install doxygen:
```Bash
sudo apt-get install doxygen
```

Run doxygen:
```Bash
doxygen doxygenConfig
```


### Using Doxygen with a GUI

Install doxywizard:
```Bash
sudo apt-get install doxygen-gui
```

Run doxywizard:
```Bash
doxywizard doxygenConfig
```
